# AT Designs

Sitio web del estudio **AT Designs**, dedicado al diseño y desarrollo de sitios web.

Página única (one page), autocontenida en `index.html`, con secciones de servicios, trabajo, proceso, nosotros y contacto.

## Características
- Diseño minimalista en paleta blanco / negro / arena.
- Tipografía Sora + Space Mono.
- Responsive (menú hamburguesa en móvil) y con animaciones suaves al hacer scroll.
- Botones de contacto por correo y WhatsApp.

## En vivo
https://at-designs.vercel.app

## Publicar
```bash
VT="$(cat ~/.config/propiedash/vercel_token)" python3 deploy.py
```
